# Lógica de Programación con PSeInt

Este repositorio contiene mis primeros pasos en el mundo del desarrollo, enfocándome en la resolución de problemas lógicos mediante pseudocódigo.

## Ejercicios Destacados
*   **Conversores de Bases:** Algoritmos para transformar números a Binario, Hexadecimal y Octal.
*   **Matemáticas:** Cálculo de Factorial y serie de Fibonacci.
*   **Gestión:** Algoritmo de cálculo de sueldo y áreas.
